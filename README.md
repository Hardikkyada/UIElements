# UIElements

In this App make Registration System with all poosible UIelement in xcode Swift


## Usage Elements
- UILabel
- UITextField
- UITextView
- UIButton
- UIDatePicker
- UIPageControl
- UISegmentedControl
- UISlider
- UIStepper
- UISwitch
- UIActivityIndicatorView
- UIImageView
- UIProgressView
- UIPickerView
- UIToolbar
- UITabBar
- UIImagePickerController
